# 版本变更

遵循语义化版本。qrels 一经冻结即不修改;后续新增或重标另开版本,评测报告须注明所依据的版本。

## v1.0(2026-06-11)

首个发布版本。

数据方面,包含 751 篇结构化案件(上海·买卖合同纠纷·2022),经规则层与大模型层抽取;在此之上建立 53 叶子、七层的裁判规则级争议焦点细谱系;按主叶子分层、判别力优先选出 58 条查询,含 19 个表里不一的陷阱案,并改写成自然语言问句。标准答案覆盖 43,558 个查询-候选对,grade 取 0–3;类案(grade≥2)共 1,370,其中 grade 3 为 351、grade 2 为 1,019。

方法方面,采用 judge-v2(问句对案件)逐对打分,每对自我一致性十次(温度 0.9),主判官为 deepseek-chat;类案带再用 deepseek-v4-pro 复判,连续分取两模型均值。每条标注含离散主档、连续期望档与不确定度。

验证方面,跨模型加权 κ 为 0.59,相邻一档以内一致率 93%,连续分相关 0.65,大致相当于人际水平;BM25 基线 NDCG@10 为 0.61(原文)与 0.72(焦点评析),落在随机与理想之间;58 个查询中 54 个的来源案连续分不低于 2.5。

已知局限见 README 相应章节。其中 Q-010(数量短缺)因属稀有薄叶子、该争点全库仅一两篇,集成后无共识类案,只计入 NDCG,二值指标按 IR 惯例跳过;58 个查询中 57 个有类案,每查询类案数中位 10、均值 24。

后续工作包括补充人工 κ(锚样本已备 150 对)、接入语义检索与精排基线、合成完整效力字段,以及扩展到民间借贷等其他案由。
